1、环境搭建：
			需要安装expect库文件
			我用的是ubuntu14.04系统，执行的命令是：apt-get install expect
2、脚本运行：
			(1)执行：chmod 777 ssh.sh scan.sh
			(2)运行: ./scan.sh

功能：这个shell脚本主要用来批量处理ssh，比如同时登陆多台主机，发送控制命令，可以登录爆破，但是只能将爆破成功的信息打印在终端上，不能记录爆破成功的信息到文本里。所以我弃用了这种方法，但是这个脚本以后可能有用，故留之。我发现还有一个登录ssh的方式：利用python开源库pexpect这个是可以实现的，但还未进行尝试，主要是pexpect安装有问题，还有待解决。
